# ZMore
```text
ZMore，纸墨收藏，根据wechaty消息来存储，复制收集指定类型文章，增强微信本身收藏能力

通过向指定账号转发消息，来Trace这些消息的站点，完成个人收藏更强大的搜索引擎
```

# How to use
```text
yarn
yarn dist
node dist/zmore.production.min.js
```

# How to develop
```text
yarn start
yarn ms
```
